Backup File Organizer

Keeps your project directories clean by automatically moving Autodesk® Revit® backup files into a separate folder.
Description General Usage Instructions Screenshots Installation/Uninstallation Contact Version History

Description

With this add-in installed, each time a backup file is created by Revit it is automatically moved into a Backups folder in the same directory.

 

Options:

  • Specify the maximum number of backups per project that you want to keep.
  • Specify the backup folder name.
  • Optionally turn on or off for project files, families & templates.

General Usage Instructions

To use, simply open a project and save it. When a new backup is created the add-in will automatically move it to a Backups folder.

Installation/Uninstallation

The installer that ran when you downloaded this app/plug-in from Autodesk App Store will start installing the app/plug-in. OR, simply double-click the downloaded installer to install the app/plugin.

To uninstall this plug-in, exit the Autodesk product if you are currently running it, simply rerun the installer, and select the "Uninstall" button. OR, click Control Panel > Programs > Programs and Features (Windows 7/8.1/10) and uninstall as you would any other application from your system.
