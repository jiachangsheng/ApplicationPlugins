Description

ElementSelector Pro is a free Autodesk® Revit® application that enables users to efficiently filter, preview, and select elements within various scopes such as an active view, a specific level, the current selection, or the entire project. This versatile tool allows users to narrow down their selections based on element category and parameter values. Furthermore, ElementSelector Pro offers several additional convenient features, including:

  • seamless interaction with elements without the need to close the dialog.
  • Optional category and parameter filters to refine the selection process.
  • Support for filtering parameters based on all parameter types.
  • The ability to simply click on a preview item (elementid) to zoom in and focus on the associated element.

General Usage Instructions

1. Select an item from the Selection Type dropdown:

  • Active View
  • By Level 
  • Current Selection
  • Entire Project

   if the selection was "By Level", another dropdown will display below to select a level.

2. Select a category from the category dropdown which displays all the related categories from the Selection Type value.

3. Select a parameter from the parameter name dropdown which displays all the related parameters from Category value.

4. Select a parameter filter rule. Just for info, there are two types of filter rules: (1) one for the string type and (2) another type for ElementId,

    Integer and Double. For the second type some rules such as: contains, does not contains, begins with, end with... will not be included at all.

5. Select a parameter value from the parameter value dropdown which displays all the related parameter values from the elements.

Installation/Uninstallation

The installer that ran when you downloaded this app/plug-in from Autodesk App Store will start installing the app/plug-in. OR, simply double-click the downloaded installer to install the app/plugin.

You may need to restart the Autodesk product to activate the app/plug-in.

To uninstall this plug-in, exit the Autodesk product if you are currently running it, simply rerun the installer, and select the "Uninstall" button. OR, click Control Panel > Programs > Programs and Features (Windows 8.1/10/11) and uninstall as you would any other application from your system.
